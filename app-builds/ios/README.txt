iOS PACKAGE INSTRUCTIONS
========================

The iOS package must be generated from PWABuilder.com:

1. Go to https://www.pwabuilder.com
2. Enter: https://www.racketrescue.com
3. Click "Start"
4. Click "Package for stores"
5. Select "iOS"
6. Configure:
   - Name: Racket Rescue
   - Bundle ID: com.racketrescue.app
7. Download and unzip here

Then open in Xcode to build and submit.

See README-VIKRAM.md for complete instructions.
